# Notebooks

The final workflow is split into three notebooks:

1. `01_task1_eda.ipynb` - data extraction, cleaning, EDA, stationarity, risk metrics.
2. `02_task2_forecasting_models.ipynb` - chronological split, ARIMA/SARIMA baseline, LSTM model, model comparison.
3. `03_task3_4_5_forecast_portfolio_backtest.ipynb` - future forecast, portfolio optimization, efficient frontier, and backtesting.

The notebooks call reusable code from `src/`, so the project remains modular and reproducible.
